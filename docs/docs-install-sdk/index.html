CortexSDK

Para gerar o SDK de Frontend basta utilizar o script build.bat localizado na seguinte diretório: SDK/src/modules/Frontend.
Será disponibilizado o arquivo cortexsdk-frontend-1.0.0.tgz no seguinte diretório: Frontend/ecommerce-app/libs

Para gerar o SDK de Backend basta utilizar o script build.bat localizado na seguinte diretório: SDK/src/modules/Backend.
Será disponibilizado o arquivo Backend-CortexSDK-1.0.0.jar no seguinte diretório: Backend/CORTEX/libs
